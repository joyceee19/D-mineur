PROJET DÉMINEUR
"Observation, Concentration, Réflexion"

Quel est le but du démineur ?
Son objectif est de découvrir l'ensemble des cases libres sans faire exploser les mines.
ATTENTION ! => Si vous touchez une seule mine, vous perdez la partie.


DÉMARRAGE 

Voici les commandes à effectuer afin de compiler le projet, lancer et faire fonctionner le programme.

1.
2. gcc -o demineur demineur.c
3. ./demineur

Lorsque vous aurez effectué ces 3 commandes, le démineur va s'afficher.
Une question va vous être posée, il faudra répondre par 1 ou 2, puis saisir les coordonnées de la case que vous souhaitez voir !
Le jeu se terminera lorsque vous aurez trouvé l'ensembles des cases libres.

Dans l'affichage:  
- le chiffre "1" signifie que vous avez trouvé une bombe,
- le chiffre "0" signifie que c'est une case libre, 
- puis, la lettre "X" signifie que vous avez placé une bombe.
